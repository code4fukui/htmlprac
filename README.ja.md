# ES-Jam: Webプログラミング道場 (htmlprac)

Monaco Editorを採用した、HTMLの学習と実践のためのシンプルなブラウザベースのエディタです。

## デモ

ブラウザで試す: **[https://code4fukui.github.io/htmlprac/](https://code4fukui.github.io/htmlprac/)**

## 機能

- **コードエディタ**: Monaco Editorを採用し、HTMLのシンタックスハイライトに対応。
- **オンデマンドプレビュー**: 「見てみる」ボタンをクリックするか`Ctrl+S`を押すと、横に並んだプレビュー画面にHTMLがレンダリングされます。
- **ファイル操作**: 作成したコードをタイムスタンプ付きの`.html`ファイルとして保存したり、ローカルのファイルをエディタに読み込むことができます。
- **自動修正**: コードのプレビュー時に、よくあるHTMLのミス（全角文字を半角文字に変換するなど）を自動的に修正します。
- **URLからの読み込み**: クエリパラメータを使用して、外部ファイルからエディタにコードをあらかじめ読み込むことができます。
- **URL経由での共有**: コードスニペットをURLハッシュにエンコードすることで、直接共有できます。

## 使い方

1. **[デモページ](https://code4fukui.github.io/htmlprac/)を開きます。**
2. **コードの記述**: 左側のエディタパネルにHTMLを入力します。
3. **プレビュー**: 「**見てみる**」ボタンをクリックするか`Ctrl+S`を押して、右側のパネルにレンダリング結果を更新・表示します。
4. **保存 / 読み込み**:
    - 「**SAVEする**」をクリックすると、コードを`.html`ファイルとしてダウンロードします。
    - 「**LOADする**」をクリックすると、ローカルの`.html`ファイルをエディタで開きます。

### URL経由でのコード読み込み

URLパラメータを使用して、エディタにコードをあらかじめ入力しておくことができます。

- **外部ファイルからの読み込み:**
    `url`クエリパラメータを使用して、プレーンテキストやHTMLファイルからコードを取得して読み込みます。
    ```
    https://code4fukui.github.io/htmlprac/?url=URL_TO_YOUR_RAW_HTML_FILE
    ```

- **URLハッシュからの読み込み:**
    HTMLをURLのハッシュに直接埋め込むことで、短いスニペットを共有できます。
    ```
    https://code4fukui.github.io/htmlprac/#<h1>Hello%20from%20URL!</h1>
    ```
