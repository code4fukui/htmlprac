# Webプログラミング 道場

Webプログラミングの学習ができるブラウザベースのエディタ環境です。

## デモ
[index.html](https://code4fukui.github.io/htmlprac/) をブラウザで開くことで、デモを確認できます。

## 機能
- コードエディタによるHTMLの編集
- 即時プレビュー
- コードの保存/読み込み

## 使い方
1. "[index.html](https://code4fukui.github.io/htmlprac/)" をブラウザで開きます。
2. エディタ部分でHTMLコードを編集すると、右側のプレビューが即座に更新されます。
3. 「SAVEする」ボタンでHTMLファイルを保存できます。
4. 「LOADする」ボタンからHTMLファイルを読み込むことができます。
